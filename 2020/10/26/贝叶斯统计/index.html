贝叶斯公式

贝叶斯公式的密度函数形式

回顾贝叶斯公式

例

例1

设某一工厂有甲、乙、丙三个车间生产同一型号的螺钉，每个车间的产量分别占该厂总产量的$25\%、35\%、40\%$，每个车间次品率分别为$5\%、4\%、2\%$。如果从全厂总产品中抽取一个螺钉，检验为次品，试问该次品是甲车间生产的概率是多少？

例二

飞机上装有警告灯，如果起落架未能完全落下，则警告灯会亮。有时，起落架完全落下了，警告灯也会亮起，现用$A_1$表示“起落架落下”，$A_2$表示“起落架未能落下”，令$B$表示“警告灯亮”，现知$P(B|A_2) = 0.999$，$P(B|A_1) = 0.005$。记录表明$P(A_1) = 0.997$，求$P(A_1|B)$。

其中，$\pi(\theta_i)$为先验分布，$\pi(\theta_i|x)$为后验分布。

$\pi$表示主观概率，$x$表示样本。

  1. 设总体指标$X$有依赖于参数$\theta$的密度函数。

    1. 在经典统计学中常记为$P(x;\theta)$或$P_\theta(x)$，它表示在参数空间$\Theta=\{\theta\}$中不同的$\theta$对应不同的分布。
    2. 在贝叶斯统计中记为$P(x|\theta)$，表示在随机变量$\theta$给定某个值，总体指标$X$的条件分布。

  2. 研究参数$\theta$的先验信息确定先验分布$\pi(\theta)$。

  3. 从贝叶斯观点看，样本$x=(x_1,x_2,\dots,x_n)$的产生要分两步进行。

    1. 首先设想从先验分布$\pi(\theta)$产生一个样本$\theta’$，这一步“老天爷”做的，人们是看不到的。

    2. 第二步是从总体分布$P(x|\theta’)$产生一个样本$x=(x_1,x_2,\dots,x_n)$，这个样本是具体的，人们能看得到的，此样本发生的概率是与如下联合密度成正比

  这个联合密度函数是综合了总体信息和样本信息，称为似然函数。

  1. 由于$\theta’$是设想出来的，它仍然是未知的，它是按先验分布$\pi(\theta)$而产生的，要把先验信息进行综合，不能只考虑$\theta’$而应对$\theta$的一切可能加以考虑。故要用$\pi(\theta)$参与进一步综合。此时$h(x,\theta) = p(x|\theta)\pi(\theta)$。把三种可用的信息都综合进去了。

共轭先验分布

定义

设$\theta$是总体分布中的参数（或参数向量），$\pi(\theta)$是$\theta$的先验密度函数，假如由抽样信息算得的后验密度函数与先验密度函数有相同的形式，则称$\pi(\theta)$是$\theta$的共轭先验分布。

例

正态均值（方差已知）的共轭先验分布是正态分布。

设$x_1,x_2,\dots,x_n$是来自正态分布$N(\theta,\sigma^2)$的一组样本观察值，其中$\sigma^2$已知。又知$\theta$的先验分布也是正态分布$N(\mu,\tau^2)$，求$\pi(\theta|x)$。

解：先验信息$\pi(\theta) = \frac{1}{\sqrt{2\pi}\tau}e^{-\frac{(\theta-\mu)^2}{2\tau^2}}$

似然函数$P(x|\theta) = \prod_{i=1}^n P(x_i|\theta) = (\frac{1}{\sqrt{2\pi}\tau})^ne^{-\frac{\sum_{i=1}^n(x_i-\theta)^2}{2\sigma^2}}$

三种信息综合

如何求$m(x) = \int h(x, \theta)d\theta$是此例的关键

记$\sigma_0^2 = \frac{\sigma^2}{n}, A = \frac{1}{\sigma_0^2}+\frac{1}{\tau^2},B = \frac{\overline{x}}{\sigma_0^2}+\frac{\mu}{\tau^2}, C = \frac{\sum_{i=1}^n(x_i)^2}{\sigma^2}+\frac{\mu^2}{\tau^2}$，则$h(x,\theta) = k_1\exp^{-\frac{1}{2}[A\theta^2-2B\theta+C]} = k_2\exp^{-\frac{(\theta-B/A)^2}{2/A}},k_2=k_1\exp^{-\frac{1}{2}(C-\frac{B^2}{A})}$

故，$m(x) = \int h(x,\theta)d\theta = k_2(\frac{2\pi}{A})^{1/2}$

则，$\pi(\theta|x) = (\frac{2\pi}{A})^{-1/2}\exp^{-\frac{(\theta-B/A)^2}{2/A}}\sim N(\mu_1,\tau_1^2)$

$\mu_1 = \frac{B}{A} = \frac{\overline{x}\sigma_0^{-2}+\mu\tau^{-2}}{\sigma_0^{-2}+\tau^{-2}},\frac{1}{\tau_1^2} = \frac{1}{\sigma_0^2}+\frac{1}{\tau^2}$

缺点

在贝叶斯统计中先验分布的选取应以合理性作为首要原则，计算上的方便与先验的合理性相比那还是第二位的。当样本均值与先验均值较远时，看来后验分布有两个峰才更为合理，可使用共轭先验分布逼使后验分布只有一个峰，从而会掩盖实际情况，引起误用，在考虑到先验的合理性之后，充分发挥共轭先验分布吸引人们的性质是我们采取的策略。

常见的共轭先验分布

共轭先验分布的选取是由似然函数$L(\theta)=p(x|\theta)$中所含$\theta$的因式所决定的，即选与似然函数具有相同核的分布作为先验分布。若此想法得以实现，那共轭先验分布就产生了。

后验均值的这一特征后被推广到单参数分布族。MacEachern指出：在单参数指数族场合，使用共轭先验分布得后验均值一定取值于先验均值与样本均值之间。
