数量生态学与R语言

R语言 生态学
发布日期:   2020-09-07

绪论

数据集

Doubs鱼类数据集

该数据集一共包括$5$个数据框，第一个数据框是$27$种鱼类在每个样方的多度，第二个数据框包括$11$个与河流的水文、地形和水体化学属性相关的环境变量，第三个数据框是样方的地理坐标（笛卡尔坐标系，$X$和$Y$）。笛卡尔坐标系通过如下过程获得。用GPS获得取样点的经纬度，然后再用R里面SoDA程序包中geoXY()函数将经纬度转为平面的笛卡尔坐标系。上面三个数据已经作为新数量方法的检验案例。另外，这里我们还附加两个材料：latlong包含有取样点的经纬度坐标，fishtraits包含描述鱼的功能属性的$4$种连续变量和$6$中二元变量。

甲螨数据集

R里面vegan和ade4包均附带这些数据。

甲螨数据集(mite.RData)也由$3$个文件构成，分别为包含$35$个形态种多度数据、$5$个基质和微地形环境数据和$70$个取样点笛卡尔坐标($x-y$)(单位：$cm$)。

探索性数据分析

目标

探索性分析(EDA)其目的在于：

  • 了解数据概况
  • 对某些变量进行转化或重新编码
  • 确定下一步分析的方向

本章以Doubs鱼类数据为例，演示R基础程序包内与探索性数据分析相关的函数，主要内容有：

  • 学习或修改R的一些基础函数
  • 学习可以应用于生态学多维数据的探索性分析技术
  • 以Doubs数据集内的水文数据为例，演示探索性数据分析过程

物种数据：第一次接触

首先要关注的是物种群落数据(Doubs.RData文件中spe对象)。由于Verneaux使用半量化的物种多度标度($0\sim5$级)，所以物种之间的多度比较是有意义的。然而，需要注意的是，这种半量化的多度标度不能被理解为真实多度（个体数量或密度）或生物量的无偏估计。

物种数据：进一步分析

上面的命令只展示数据的大体结构。但这些代码和数值看起来并不吸引人，也没有启发性，本小节展示数据的一些其他属性。我们通过代码生成样方位置分布图。

#样方位置地图
#生成空的绘图底板（横纵坐标轴比例1:1(参数asp),带标题）
#从spa数据框获取地理坐标x和y
#spa是样方的地理坐标（笛卡尔坐标系）
plot(spa, asp = 1, type="n", main="样方位置", xlab="x坐标(km)", ylab="y坐标(km)")
#加一条连接各个样方点的蓝色线（代表Doubs河）
lines(spa, col="light blue")
#添加每个样方的编号
text(spa, row.names(spa), cex = 0.8, col = 'red')
#添加文本
text(70, 10, "上游", cex = 1.2, col = "red")
text(20, 120, "下游", cex = 1.2, col = "red")

当数据集覆盖足够大的区域时，可以投影采样点到GoogleMaps地图上：

#将样方点映射到谷歌地图
#以googleVis程序包默认的方法组织数据
#从浏览器中生成图
nom <- latlong$Site
latlong2 <- paste(latlong$LatitudeN, latlong$LongitudeE, sep = ":")
df <- data.frame(latlong2, nom, stringsAsFactors = FALSE)#传给gvisMap函数数据框的格式
mymapl <- gvisMap(df, locationvar = "latlong2", tipvar = "nom", options = list(showTip = TRUE))#locationvar指地理坐标,tipvar指每个点的标签
plot(mymapl)
