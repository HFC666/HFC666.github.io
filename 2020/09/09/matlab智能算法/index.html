符号矩阵的生成

用命令sym定义矩阵

这时的函数sym实际上在定义一个符号表达式，符号矩阵中的元素可以是任何符号或者表达式，而且长度没有限制，只是将方括号置于用于创建符号表达式的单引号中。

经过实践，由此方法创建符号矩阵已经不行了，改为用str2sym函数

>> sym_m = str2sym('[a,b,c;Jack,Me,WAY]')

sym_m =

[    a,  b,   c]
[ Jack, Me, WAY]

>> sym_d = str2sym('[1 2 3;a b c;sin(x) cos(y) tan(z)]')

sym_d =

[      1,      2,      3]
[      a,      b,      c]
[ sin(x), cos(y), tan(z)]

用命令syms定义矩阵

先定义矩阵中的每一个元素为一个符号变量，然后如数值矩阵操作那样输入符号矩阵。

>> syms a b  c
>> M1 = sym("Classical");
>> M2 = sym("Claysw");
>> M3 = sym("Yellow");
>> yswM123 = [a b c;M1 M2 M3;2 3 4;5 4 6]

yswM123 =

[         a,      b,      c]
[ Classical, Claysw, Yellow]
[         2,      3,      4]
[         5,      4,      6]

把数值矩阵转化成相应的符号矩阵

数值型和符号型在MATLAB中是不相同的，它们之间不能直接进行转化。MATLAB提供了一个将数值型转化为符号型的命令，即sym。

>> Digit_Ma = [1/3 sqrt(3) 3.1; exp(0.3) log(10) 23^0.5];
>> Syms_Ma = sym(Digit_Ma)

Syms_Ma =

[                               1/3,                           3^(1/2),    31/10]
[ 3039611811401035/2251799813685248, 2592480341699211/1125899906842624, 23^(1/2)]

一般情况下，矩阵是以浮点型变量保存的。矩阵转化为符号矩阵后以最接近原值的有理数形式或者函数形式保存。

友矩阵

函数compan

格式 A = compan(u) $\%u$为多项式系统向量，$A$为友矩阵，$A$的第一行元素为$-u(2:n)/u(1)$，其中$u(2:n)$为$u$的第$2$到第$n$个元素，$A$的特征值就是多项式的特征根

线性方程组求解

齐次线性方程组的通解

在MATLAB中，函数null()用来求解零空间，即满足$A\cdot X=0$的解空间，实际上是求出解空间的一组基。

  • z = null ：$z$的列向量为方程组的正交规范基，满足$Z^{‘}\times Z=I$
  • z = null(A, 'r')：$z$的列向量是方程$A\cdot X=0$的有理基
>> A = [1 2 2 1;2 1 -2 -2;1 -1 -4 -3]; %原始系数矩阵
>> format rat
>> B = null(A, 'r')

B =

       2              5/3     
      -2             -4/3     
       1              0       
       0              1

或通过最简行得到基：

>> B = rref(A)

B =

       1              0             -2             -5/3     
       0              1              2              4/3     
       0              0              0              0

则相应地写出线性方程组的通解：

>> X = k1*B(:,1)+k2*B(:,2)

X =

   2*k1 + (5*k2)/3
 - 2*k1 - (4*k2)/3
                k1
                k2

>> pretty(X)
/         5 k2  \
|  2 k1 + ----  |
|           3   |
|               |
|          4 k2 |
| - 2 k1 - ---- |
|            3  |
|               |
|       k1      |
|               |
\       k2      /

非齐次线性方程组的通解

需要先判断非齐次线性方程组是否有解。若有解，然后求通解，步骤如下。

  1. 判断$A\cdot X=b$是否有解。若有解，则进行第二步，否则终止求解
  2. 求$A \cdot X=b$的一个特解。
  3. 求$A\cdot X=0$的通解
  4. $A \cdot X=b$的通解等于$A\cdot X=b$的通解加上$A\cdot X=b$的一个特解

求解方程组的解

A = [1 -2 3 -1;3 -1 5 -3;2 1 2 -2];
b = [1 2 3]';
B = [A b];
n = 4;
RA = rank(A);
RB = rank(B);
format rat %有理数格式
if RA==RB&&RA==n
    x=A\b;
elseif RA==RB&&RA> X =

    'equition no solve'
